# C/C++ Coding Standards

## Bracing Style
* **Always** use **Allman style** bracing (braces on a new line).
* Do not use K&R style (braces on the same line).

## Examples
// Good
if (condition) 
{
    // code
}

// Bad
if (condition) {
    // code
}

## Indentation Style
* Use 4 spaces
